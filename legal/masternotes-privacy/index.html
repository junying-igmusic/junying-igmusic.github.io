    1. 資料控制者

    炎上音樂科技有限公司（「本公司」、「我們」）是一家在香港特別行政區成立的科技公司，致力於為全球古典音樂社群提供數碼化解決方案。本公司為大師說（Masternotes）應用程式及相關服務的資料控制者。

    如您對本政策有任何疑問，或希望行使您的資料權利，請電郵至 legal-masternotes@igmusictech.com 與我們聯繫。

    資料說明法律依據（GDPR）

    2.3 AI 自動建立的用戶畫像

    在您的對話過程中，我們的 AI 會自動提取並儲存有關您音樂偏好、品味及對特定作品或藝術家熟悉程度的相關事實——此過程無需您進行任何手動操作。這些推斷出的事實將用於個性化您未來的對話。此舉構成 GDPR 第 22 條所界定的自動化分析，您有權提出反對。

    欄位內容示例
    fact（事實）偏好描述偏好布倫德爾的錄音
    category（類別）preference（偏好）/ taste（品味）/ familiarity（熟悉度）/ other（其他）preference
    confidence（置信度）AI 置信分數 0.0–1.00.85

    反對權：您可隨時發送電子郵件至 legal-masternotes@igmusictech.com，要求刪除所有已推斷的個人資料。我們將在 30 天內予以回應。

    4. 第三方數據處理者

    您的訊息內容將被傳輸至以下 AI 服務商以生成回應。每家服務商均依據數據處理協議（DPA）作為數據處理者。我們不會將您的資料出售給任何第三方。

    服務商服務傳輸的資料隱私政策
    OpenAILLM 及嵌入向量用戶訊息內容openai.com/policies/privacy
    AnthropicLLM 對話（Claude）用戶訊息內容anthropic.com/privacy
    Google GeminiLLM 對話（Gemini Flash）用戶訊息內容policies.google.com/privacy
    Supabase資料庫、身份驗證、儲存所有用戶及內容資料supabase.com/privacy
    Wikimedia Foundation（維基百科）透過 Wikipedia API 進行即時知識搜尋AI 生成的搜尋關鍵詞（不含帳號資料）foundation.wikimedia.org/wiki/Privacy_policy

    Wikipedia API 使用說明：當您的查詢所需資訊無法從我們的內部知識庫中獲取時，我們的 AI 可能會將從您對話中衍生出的搜尋關鍵詞傳送至 Wikipedia API（由 Wikimedia Foundation 運營），以檢索相關百科內容。傳輸的僅限搜尋關鍵詞，不包含任何帳號識別資訊或個人資料。Wikipedia 內容依據 Creative Commons 姓名標示-相同方式分享 4.0（CC BY-SA 4.0）授權；應用程式始終向用戶提供來源鏈接。

    7. 最低年齡

    本服務適用於 16 歲及以上的用戶。我們不會故意收集 16 歲以下人士的個人資料。若您認為我們已收集了未成年人的資料，請立即通過 legal-masternotes@igmusictech.com 與我們聯繫，我們將立即予以刪除。

    8. 資料安全

    我們採取適當的技術及組織措施，包括：對所有用戶資料表實施行級安全（RLS）；JWT 令牌儲存於 iOS Keychain；加密密碼儲存；基於路徑的上傳文件隔離儲存；伺服器端記憶體快取於重啟時清除。

    9. 國際資料傳輸

    當您的訊息由 OpenAI、Anthropic 或 Google 處理時，資料可能會被傳輸至美國並在當地進行處理。此類傳輸受各服務商包含歐盟標準合約條款（SCC）的 DPA 約束。

    10. 未來功能及第三方登錄

    我們未來可能推出以下功能。在推出時，本政策將予以更新，並提前至少 14 天通知用戶。

    10.1 應用程式內購及訂閱

    我們可能推出透過 Apple 應用程式內購基礎設施處理的付費訂閱方案。訂閱相關資料（方案類型、續訂狀態）將儲存於您的帳號中。我們不會接收或儲存支付卡資訊——所有付款均由 Apple 處理。

    10.2 第三方登錄

    我們可能提供第三方身份驗證選項。依據 Apple App Store 指引，在 iOS 設備上，「以 Apple 帳號登入」將始終與其他社交登入選項一同提供。

    服務商接收的資料備注
    以 Apple 帳號登入姓名（選填，由用戶控制）、電子郵件或 Apple 中繼電子郵件、唯一 Apple IDApple 可能以中繼地址取代真實電子郵件
    其他社交登入（待定）姓名、電子郵件地址、個人資料圖片、服務商用戶 ID僅基本個人資料；不存取聯絡人、帖子或其他帳號資料

    從第三方登錄服務商接收的所有資料均儲存於 Supabase Auth，並受本政策所述的相同保護措施約束。您可隨時透過 Apple ID 設定或相應服務商的帳號設定撤銷第三方登錄存取權限。

    10.3 使用分析

    我們未來可能整合使用分析功能，以改善 AI 個性化及產品體驗。如予引入，分析服務商將在本政策中予以披露，我們將向您提供適當通知，並在法律要求時提供退出選項。

    11. 政策變更

    我們可能不時更新本隱私政策。重大變更將透過應用程式內通知或電子郵件傳達，並在生效前至少 14 天發出通知。最新版本始終可在以下網址查閱：https://igmusictech.com/legal/masternotes-privacy/

    12. 適用法律及聯絡方式

    本隱私政策受香港特別行政區法律管轄，包括《個人資料（私隱）條例》（第 486 章）。對於歐盟/歐洲經濟區及英國用戶，GDPR 及英國 GDPR 亦額外適用。

    8. Data Security

    We implement appropriate technical and organisational measures including: Row Level Security (RLS) on all user data tables; JWT tokens stored in iOS Keychain; encrypted password storage; path-based storage isolation for uploaded files; server-side in-memory caching cleared on restart.
